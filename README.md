# Industrial Process Manager

Application simple de gestion industrielle pour OCP.

Le projet contient :
- un frontend Vue.js ;
- un backend Java natif ;
- une connexion MySQL ;
- des espaces selon le role utilisateur.

Roles principaux :
- ADMIN : dashboard global, gestion users, gestion ateliers ;
- CHEF_ATELIER : dashboard atelier, equipements, operations, lots ;
- TECHNICIEN_LABO : acces laboratoire.
